### This project contains 👇

1. Working with multiple api versions.
2. About `@GetExchange` to demonstrate **HttpClient** auto configuration.
3. Resilient Features in Spring. `@EnableResilientMethods`, `@Retryable`, `@ConcurrencyLimit`
4. PassKey and Multi-factor Authentication.
5. `BeanRegistrar` programmatically register bean.
